Introduction

In this page, I will talk about the lessons from Week 1. In the first week of instructions, we learned a bit about Github, Sumblime Text, the Command Line, and all sorts of neat commands. We got a brief introduction into HTML, CSS, and JavaScript.

  • HTML is used for structure.
  • CSS is used for style
  • JavaScript is used for behavior

We learned more about what HTML is. HTML means "HyperText Markup Language". The word "HyperText" refers to the way texts contain links to other texts. The word "Markup" refers to the way HTML annotates content using tags, telling the browser or machine what the content is. The word "Language" refers to the syntax of HTML. This is because there is a right and wrong way to code

Before 1997, web development and design was not driven by standards. Over time standards such as XHTML 1.0 (2000) and WHATWG:HTML (2004) was introduced by a bunch of vendors that came together (Web Hypertext Application Technology Group). In 2007, WHATWG & W3C started to work together. What they produced is HTML 3. These two organisations are basically in charge of HTML. W3C has standards while WHATWG is takes the stance that it is evolving.

A CSS rule consists of 2 main parts:

  2. The second part is the declaration which is inside the brackets and it consits of two parts the property and value:

    For every property there is a set number of values such "blue" or a numerical value. The property is predefined by CSS speficification. There can zero or more declarations:

  3. The collection of the CSS Rules is called a stylesheet:

There are three types of selectors: element selector, class selector, and id selector

  1. The element selector refers to you specifyting the element selector such as "p" or "div".

  2. The class selector is specified with a dot and the name of the class. You define the class with a dot and use it without a dot.

  3. The id selector is specified with a pound sign "#" and it is the value of some id in the html document. E.g., the class speficied "#blue {color: blue;}," can be used inside a div element as an id, <div class = "blue">."

  4. Selectors can also be grouped together. Also you can use "*" to define all elements on the page.
